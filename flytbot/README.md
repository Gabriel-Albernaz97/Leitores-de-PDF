# Bot Leitor de PDF
Projeto em Python para leitura e identificação automática de PDFs fiscais.
Possui um leitor principal responsável por identificar o tipo de imposto (federais e ICMS) e direcionar o processamento para o leitor específico correto, retornando os dados estruturados em formato JSON.
O projeto também inclui um leitor para extrato mensal de folha de pagamento.
